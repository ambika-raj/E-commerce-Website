# E-Commerce Product Website

A responsive e-commerce product listing website built using **HTML, CSS, JavaScript and JSON**.  
This project fetches product data from an external API and displays it with features like search, pagination, product details, and history tracking using LocalStorage.

---

## Features

###  Product Listing
- Fetches products dynamically from API
- Displays product image, title, price, and description
- Responsive product cards UI

### Search Functionality
- Search products by title
- Redirects to search results page
- Stores search history using LocalStorage

### Product Details Page
- Click any product to view detailed information
- Fetches product data using product ID
- Saves visit history

### History Tracking
- Search history stored in browser LocalStorage
- Visit history stores clicked products with timestamp

### Pagination
- Displays products page-wise
- Dynamic page buttons
- Active page highlighting

### Responsive Design
- Works on mobile, tablet, and desktop screens
- Flexible grid layout

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- DummyJSON API
